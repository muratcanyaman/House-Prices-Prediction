
# House Prices Prediction

# House Prices 
Kaggle'daki **House Prices: Advanced Regression Techniques** yarismasi icin hazirlanan regresyon notebook'u.
## Proje Ozeti

Bu calisma su adimlardan olusur:

- temel ve gelismis EDA
- eksik veri duzenleme
- feature engineering
- leak-free cross validation
- birden fazla modelin weighted blend ile birlestirilmesi
- `submission.csv` uretimi

Notebook icinde:

- sadece code cell bulunur
- her hucre yorum satirlari ile aciklanir
- Kaggle ve GitHub'da kolay okunacak duzende tutulur

## Kullanilan Modeller

- Lasso
- ElasticNet
- GradientBoosting
- XGBoost
- CatBoost

## Dosyalar

- `house-prices-advanced-regression.ipynb`: final notebook
- `train.csv`: egitim verisi
- `test.csv`: test verisi

- Python 3.10+
